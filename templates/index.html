          HOW TO USE · TIPS · TROUBLESHOOTING
          ▶
        

        

          SETUP: Run Ollama on a machine reachable from here.

          Install a model: ollama pull llama3

          Set the URL (e.g. http://192.168.40.15:11434) and model name, then SAVE.

          Use TEST to verify connectivity and list available models.


          PERSONA: Shapes every reply. Keep it short and specific.

          Example: "You are a gruff shortwave numbers station announcer. Keep responses under 20 words. Only speak in numeric groups."


          TEMPERATURE: 0.0 = deterministic, 2.0 = wild. 0.7–1.2 recommended for radio use.


          TX NOT WORKING?

          • Check the DEBUG LOG — enable VERBOSE for full trace

          • TX runs in a background thread; check journald: journalctl -u riggpt -f

          • Ensure eSpeak is installed: which espeak-ng

          • Confirm radio is connected (Dashboard → CONNECT) for PTT; audio plays regardless

          • Try eSpeak engine — it requires no voice files and always works offline


          INSPIRE: Generates a random TX idea into the prompt box for you to review and edit before sending.

          PLAY: Synthesizes the AI response locally and plays through speakers — no PTT keyed.

          AUTO TX: Every AI response is automatically transmitted. Use carefully.
